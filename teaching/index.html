I developed or contributed to these educational materials, which range from simple cheat sheets to in-depth tutorials, and cover topics from introductory programming to molecular modeling and simulation. Feedback on existing content and suggestions for other tutorials are welcome.

Intermediate Python

A 3-hour Python workshop that aims to familiarize participants with conventions and features of the language that are not necessarily covered in introductory courses, but that are nonetheless common in daily Python programming, by building a road trip planner. Link, Source.

Intro to Conda

A short Python workshop that aims to get participants started with conda environments to manage Python packages. Link, Source.

Git Cheat Sheet

A simple Git cheat sheet for personal use or for use in Git workshops. Can be rendered at HTML or printed as PDF. Link, Source.

Molecular Modeling and Simulation

A syllabus developed for a MSc-level course on homology modeling, molecular dynamics simulation, and protein-protein docking. Originally developed by myself, partially based on existing tutorials by Tsjerk Wassenaar and Adrien Melquiond, these materials are now maintained by the Bonvin lab. Link, Source, and Data.
